🏫 Parent Cockpit — React frontend

This is the Vite/React source. To run it:

cd frontend
npm install
npm run dev         # http://localhost:3000 (proxies /api to :8000)

Node 20+ required (winget install OpenJS.NodeJS.LTS).

In the meantime, the backend already serves a full HTML Today view at http://localhost:8000/ with no Node needed.
